🦾 Orçameintor Pro | Versão 2.3Sistema para gestão de custos, precificação técnica com BDI e geração de propostas em PDF.🚀 Funcionalidades PrincipaisEngenharia de Custos: Separação entre custos diretos (materiais/itens) e custos indiretos (logística/mão de obra).Cálculo de Margem (BDI): Aplicação automática de lucro e impostos sobre o somatório de todos os custos.Modo Apresentação: Recurso visual para ocultar valores sensíveis (lucro e custos unitários) durante reuniões com clientes.Controle de Revisão: Campo para versionamento de propostas (Ex: REV 01, REV 02), facilitando o rastreio de negociações.Persistência de Dados: Salvamento automático no navegador e opção de exportar o projeto em arquivo para backup.🧮 Lógica do CálculoA fórmula aplicada pelo sistema segue o padrão de faturamento de serviços:$$Total = [ (Produtos) + (Dias \times Equipe \times Diária) + Gasolina + Alimentação ] \times Fator$$O Fator é a combinação da Complexidade, Margem de Lucro e Impostos.🛠️ Como UtilizarIdentificação: Insira o nome do cliente e do profissional responsável.Escopo: Adicione os materiais ou serviços. Use o preço de custo.Logística: Defina o prazo da obra, o tamanho da equipe e os custos diários/totais de operação.Financeiro: Ajuste a margem de lucro e os impostos previstos para a emissão da nota fiscal.Emissão: Gere o PDF. O sistema agrupa automaticamente os custos de logística sob o termo "Mobilização e Mão de Obra Técnica" para manter a proposta profissional.📁 BackupUse o botão Salvar Projeto para baixar um arquivo com todos os dados preenchidos. Isso permite que você retome o orçamento em outro computador ou no futuro.
